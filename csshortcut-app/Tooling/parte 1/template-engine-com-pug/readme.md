# PUG

É um motor de template (template engine) de alto desempenho para Node.js e navegadores, que foca em uma sintaxe limpa e sensível a espaços em branco para escrever HTML. Ele substitui a verbosidade do HTML tradicional por uma estrutura baseada em indentação, eliminando a necessidade de fechar tags manualmente.

Documentação oficial: https://pugjs.org/api/getting-started.html
