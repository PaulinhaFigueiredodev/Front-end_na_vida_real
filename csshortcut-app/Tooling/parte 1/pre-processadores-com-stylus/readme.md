# Pré processador CSS

Um pré-processador CSS é uma linguagem que estende o CSS com recursos que ele não tem nativamente, como:

- Variáveis
- Aninhamento de seletores
- Mixins / funções reutilizáveis
- Operadores matemáticos
- Herança e funções

Ao final, todo o código é transformado em CSS comum pelo compilador.

## Stylus

Stylus é um pré-processador de CSS — uma linguagem que adiciona funcionalidades ao CSS tradicional para deixar o desenvolvimento de estilos mais rápido, mais organizado e mais poderoso, e depois compila (pré-processa) para CSS que o navegador entende.

### O que o Stylus oferece?

O Stylus é um dos pré-processadores populares (junto com Sass/SCSS e Less). Ele se destaca por ser bem flexível e permissivo — permite escrever sem ponto e vírgula, parênteses e chaves se você quiser.
